# Claude Code Scaffolding

Personal templates and workflow documentation for projects built with Claude Code.

- **`new_project_scaffold.md`** — Paste into Claude Code at the start of any
  new project. Sets up `/docs`, `.claude/agents/`, and `.claude/commands/`
  with my standard 11-doc, 7-agent, 11-command structure.

- **`DAILY_PLAYBOOK.md`** — Daily-use reference for the workflow once a
  project is up and running. Covers the standard session loop, common
  scenarios, anti-patterns, and a worked example.

These work as a pair: the scaffold sets up the infrastructure, the playbook
explains how to use it day-to-day.
